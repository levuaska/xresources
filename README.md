<h2>Installation</h2>
  
- Use curl:
  ```
  curl -o xresources https://raw.githubusercontent.com/levuaska/xresources/main/.Xresources
  ```

- Active theme using: 
  ```
  xrdb merge <path to xresources>
  ```
